# Bio-Mask Audio Assets

Drop your sound files here to customize the HUD experience.

### Instructions:
1.  Add `.wav` or `.mp3` files to this directory.
2.  The filename (uppercase, no extension) becomes the trigger name.
3.  Restart the application to reload assets.

### Recommended Files:
- `mode_change.wav` -> Played during visor mode transitions.
- `target_lock.wav` -> Played when a target is acquired in AUTO_TARGET mode.
- `error.wav`       -> Played when telemetry is corrupted.
